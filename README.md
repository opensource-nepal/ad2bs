# ad2bs

Converts a given AD date to BS date and BS date to AD.

## Usage

**Converting AD to BS**

```bash
ad2bs YYYY-MM-DD
```

**Converting BS to AD**

```bash
bs2ad YYYY-MM-DD
```

Example:

```bash
ad2bs 1999-10-29
```

## Installation

```sh
curl -s https://raw.githubusercontent.com/aj3sh/ad2bs/main/scripts/install.sh | sh
```
